# Массив (Скамейка фиксированной длины)

 Принцип хранения:
Все элементы сидят плотно друг к другу на одной скамейке известной длины.

Мальчики: [Андрей, Борис, Василий]  
Скамейка: `[ _А_ | _Б_ | _В_ ]` (3 места)

 Что происходит:
1. Добавить Диму в конец → Есть место? Нет!
2. Ищут новую скамейку на 4+ мест
3. Все пересаживаются (Андрей, Борис, Василий, Дима)
4. Старая скамейка освобождается

Плюсы:
- Быстро найти любого: "Василий, ты на 3-м месте!"
- Все сидят рядом (быстрый доступ)

Минусы:
- Добавить/убрать в середине → все сдвигаются
- Мало мест → ищут новую скамейку (перевыделение памяти)

Операции:
- Найти по номеру места: Мгновенно (знаю где сидит)
- Добавить в конец: Обычно быстро (если есть место)
- Добавить в середину: Медленно (все сдвигаются)
- Удалить из середины: Медленно (все сдвигаются)

---

# Связанный список (Дети держатся за руки)

 Принцип хранения:
Каждый стоит где хочет, но держит за руку следующего.

```
Андрей → (держит за руку) → Борис → (держит за руку) → Василий
    ↑                            ↑                            ↑
Стоит у дерева              У фонаря                     У качелей
```

 Что происходит:
1. Добавить Диму после Бориса:
   - Дима встает где угодно (у горки)
   - Борис отпускает руку Василия
   - Борис берет за руку Диму
   - Дима берет за руку Василия

Плюсы:
- Добавить/убрать в любом месте: Быстро (только поменять "рукопожатия")
- Не нужно всем переходить на новое место

Минусы:
- Найти 3-го: Медленно (спрашиваю у Андрея: "Кто следующий?", у Бориса: "Кто следующий?")
- Дети стоят далеко друг от друга (нет локальности в памяти)

Операции:
- Найти по номеру: Медленно (иду по цепочке)
- Добавить в начало: Быстро (новый берет за руку первого)
- Добавить в середину: Быстро (меняем "рукопожатия")
- Удалить из середины: Быстро (соседи берутся за руки друг с другом)

# Сравнение в жизни

 Массив (школьный класс):
- У каждого фиксированное место за партой (№1, №2, №3...)
- Учитель быстро вызывает: "Ученик №15, к доске!"
- Нового ученика трудно впихнуть между партами
- Если класс заполнен → переезжаем в другой кабинет

 Связанный список (очередь в столовой):
- Каждый стоит где попало
- Но знает, кто стоит за ним
- Новый человек может встать в середину: "Я за тобой!"
- Чтобы найти 10-го человека, нужно спросить у 9-ти людей

---

Параметр |	Массив (List)	| Связанный список
Чтение:	 |      O(1)	           O(n)
Запись:	 |      O(n)	           O(1)


# Итог одной фразой:

Массив — все сидят по порядку на одной скамейке.  
Список — все стоят где угодно, но держатся за руки.
